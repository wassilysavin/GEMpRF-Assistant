prfprepare in GEM-pRF. The website's input-source tutorial names prfprepare as one of the two BIDS-derivative input styles GEM-pRF accepts for surface data (the other is fMRIPrep). DemoKit example 001 demonstrates BIDS + prfprepare (surface, individual runs); example 005 demonstrates fixed paths + prfprepare (surface). The corpus does not contain a deeper definition of prfprepare beyond this usage.
