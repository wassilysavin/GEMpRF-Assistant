GEM-pRF configuration docs. The website states that the XML configuration file is the heart of GEM-pRF. The public guidance says all analysis settings, paths, and parameters live in the XML, while Python usage stays small: import gemprf as gp and call gp.run(config_path). The docs recommend meaningful XML naming and treat the XML as the reproducible record of an analysis.
